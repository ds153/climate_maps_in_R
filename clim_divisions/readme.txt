The following are the U.S. climate division files in ArcView shapefile 
format. The data set was generated from the ASCII flat file that 
contains points identifying each polygon boundary.  

There are four main fields associated with each polygon:

1) st - the state abbreviation (e.g. MN for Minnesota)
2) div - the division number (e.g. 6)
3) name - division name (e.g. EAST CENTRAL)
4) division_id - comprised of state and division number (e.g. 2106 = Minnesota, division 6)

Each of the three files is necessary for ArcView.  
The files are division.shx, division.shp, division.dbf.
They are located at ftp://climate/ftp/pub/data/divboundaries/gis/
